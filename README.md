# Neonatal Health Risk Prediction

## Overview
This project focuses on predicting neonatal health risk levels using machine learning techniques based on the Data Science Life Cycle.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Logistic Regression and Decision Tree models
- Model evaluation using accuracy, precision, recall, and F1-score

## How to Run
1. Open the notebook in Google Colab
2. Upload the dataset file
3. Run all cells sequentially

## Dataset
Newborn Health Monitoring Dataset (Kaggle)
